# Drone-Detection-and-Identification
Drone Detection and Identification via Radio Frequency Fingerprinting Using HackRF One Software Defined Radio
This script generates complex In-phase (I) and Quadrature (Q) noise structures (Hypothesis $H_0$), injects transient Frequency Hopping Spread Spectrum (FHSS) drone control bursts (Hypothesis $H_1$), calculates an adaptive energy detection threshold using a Constant False Alarm Rate (CFAR) approximation, and renders a waterfall spectrogram.
